# L'Intant

## Choix de la techno
Pour le Front-end j'ai choisi d'utiliser React car très pratique au niveau de l'agencement du projet et de sa popularité (plus facile de trouver d'autres développeurs),
Pour le Back-end j'ai utilisé la bibliothèque express JS de NodeJS pour rester sur du Java Script pour l'ensemble du projet.

## A savoir 
Il manque quelques features au projet (ajout d'image, like)
Le front-end est hébergé sur Netlify, mais le back-end n'a pas pu être hébergé à cause d'une dépendance (sqlite3).
Pour démarrer l'aplication on doit donc démarrer le back-end localement :

### `cd .\server\`
### `npm i` (pour le premier démarrage)
### `npm start` ou `node index.cjs`

l'API sera donc en route sur `http://localhost:3001`
